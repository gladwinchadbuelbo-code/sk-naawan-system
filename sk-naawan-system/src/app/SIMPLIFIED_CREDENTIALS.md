## 📋 Login Instructions

1. Go to the login page
2. Enter your username (e.g., `president`)
3. Enter the same word as your password (e.g., `president`)
4. Click "Log In"

## 💡 Why This Change?

This simplification makes the demo system:
- ✅ Easier to remember
- ✅ Faster to test
- ✅ More user-friendly for demonstrations
- ✅ Consistent across all roles

## ⚠️ Important Notes

1. **Case Sensitive**: Use lowercase for all credentials
2. **No Spaces**: Don't include any spaces in username or password
3. **Minimum Length**: Each credential meets the 6-character minimum requirement
4. **Production Use**: For production deployment, implement strong passwords and proper authentication
