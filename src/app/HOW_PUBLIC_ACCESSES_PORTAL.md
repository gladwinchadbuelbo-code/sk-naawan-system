# 🌐 HOW PUBLIC VIEWERS ACCESS THE TRANSPARENCY PORTAL

## **📍 SIMPLE ANSWER:**

### **Method 1: From Homepage (Recommended)**
```
1. Open the website
2. You'll see the landing page with TWO big cards
3. Click the LEFT card that says "Public Transparency Portal"
4. ✅ You're in! No login needed.
```

### **Method 2: Direct URL**
```
Just go to: /public

Example:
- If the app is at: http://localhost:3000
- Go to: http://localhost:3000/public
```

## **🗺️ ALL 5 PUBLIC PAGES YOU CAN ACCESS**

Once you're in `/public`, you can navigate to:

### **1️⃣ Budget Transparency** (`/public/budget`)
```
💰 What's Inside:
- Total budget allocated
- Total expenses to date
- Remaining balance
- Budget breakdown by category
- All transactions with receipts
- Download budget report PDF
```

### **2️⃣ Programs & Activities** (`/public/activities`)
```
📅 What's Inside:
- List of all approved events
- Upcoming activities
- Completed activities
- Event photos and documentation
- Accomplishment reports (downloadable)
- Filter by status/date
```

### **3️⃣ Event Calendar** (`/public/calendar`)
```
📆 What's Inside:
- Monthly calendar view
- All events marked on dates
- Click event for details
- Navigate months (prev/next)
- Upcoming events list
```

### **4️⃣ Financial Records** (`/public/budget`)
```
📄 What's Inside:
- All income transactions
- All expense transactions
- Scanned receipts (view/download)
- Transaction details
- Categorized spending
```

### **5️⃣ Search & Downloads** (`/public/search`)
```
🔍 What's Inside:
- Search bar for all documents
- Filter by:
  - Document type
  - Date range
  - Category
- Download individual reports
- Download popular documents
- Advanced search features
```

## **🔐 WHAT PUBLIC CANNOT DO**

For security, public viewers **CANNOT**:
- ❌ Edit any data
- ❌ Delete anything
- ❌ Create proposals
- ❌ Upload documents
- ❌ Approve budgets
- ❌ Access staff functions
- ❌ See pending/draft proposals
- ❌ View internal comments

**They can ONLY VIEW approved, published information.**

```
2. Click "Budget Transparency"
3. See complete breakdown:
   - ₱50,000 allocated
   - ₱35,000 spent
   - ₱15,000 remaining
4. Click on any expense to see receipt
5. Download full budget report
```

### **Scenario 2: Parent**
```
Question: "What activities did SK do this year?"

Answer:
1. Go to /public
2. Click "Programs & Activities"
3. See all events:
   - Leadership Training (Completed)
   - Sports Fest (Completed)
   - Tree Planting (Upcoming)
4. Click on event to see:
   - Photos
   - Accomplishment report
   - Budget used
   - Participants

```

```
Answer:
1. Go to /public

   - Event documentation
```

```
5. Cross-reference with accomplishment reports
```
