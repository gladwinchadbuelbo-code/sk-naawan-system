## 📋 **Step-by-Step: Checking Proposal Status**

### **Quick Check (30 seconds):**

1. **Look at notification bell** 🔔
   - Badge number = new updates
   
2. **Click bell** → Read notification
   - ✅ Green = Approved
   - ↩️ Orange = Returned
   - 💬 Contains chairperson's comments

3. **Click "View My Proposals"** button in notification
   - Opens My Proposals page

---

### **Detailed Check (2 minutes):**

1. **Go to Dashboard**

2. **Click "My Proposals"** (blue button)

3. **View Summary Cards** at top
   - See counts: Pending, Approved, Returned

4. **Scroll through proposals**
   - Each card shows status badge
   - Read chairperson's comments
   - Check review dates

5. **Click "View Full Details"** for complete info
   - See full proposal
   - Timeline of submission → review
   - All comments and feedback

## 🎓 **What Happens When Approved?**

### **Automatic Actions:**

1. **✅ Proposal Status → "Approved"**
2. **📅 Event Created** in Events page automatically:
   - Title: Your proposal title
   - Date: Your proposed date
   - Location: Your venue
   - Budget: Your budget amount
   - Participants: Your target number
   - Status: "Upcoming"

3. **🔔 You Get Notification:**
   ```
   ✅ Activity Proposal Approved
   Your proposal "Youth Summit 2025" has been approved.
   Comment: [Chairperson's message]
   ```

4. **📝 Activity Logged:**
   ```
   [Chairperson Name] approved activity proposal: [Your Title]
   ```

---

## ↩️ **What Happens When Returned?**

### **Actions Needed:**

1. **↩️ Proposal Status → "Returned"**

2. **🔔 You Get Notification:**
   ```
   ↩️ Activity Proposal Returned
   Your proposal "Sports Fest" has been returned for revision.
   Comment: [Chairperson's specific feedback]
   ```

3. **💬 Read Comments** carefully:
   - Chairperson explains what needs changing
   - Specific items to fix

4. **✏️ Options:**
   - Click **"Create Revised Proposal"** button
   - Opens new proposal form
   - Pre-fill with corrected information
   - Submit again as new proposal

## 💡 **Pro Tips for Secretary**

### **1. Check Regularly**
- **Daily**: Check notification bell (🔔)
- **After submission**: Check My Proposals page

### **2. Respond Quickly to Returns**
- Read comments carefully
- Fix issues mentioned
- Resubmit within 24-48 hours

### **3. Keep Good Records**
- Screenshot approved proposals
- Save chairperson's comments
- Track approval timeline

### **4. Plan Ahead**
- Submit proposals 2+ weeks before event
- Allow time for revision if needed
- Don't wait for urgent deadlines

## 🚨 **Troubleshooting**

### **Q: I submitted a proposal but don't see it in My Proposals**
**A:** 
1. Refresh the page
2. Check if you're logged in as Secretary
3. Check Activity Log for submission record
4. Ensure proposal was successfully submitted (check for success toast message)

---

### **Q: Chairperson approved but I don't see notification**
**A:**
1. Refresh the page
2. Check notification bell (🔔) - click it
3. Go to My Proposals page - status should show ✅ Approved
4. Check Events page - event should appear

---

### **Q: How do I know if chairperson has seen my proposal?**
**A:**
- No direct "read receipt" yet
- Check My Proposals page:
  - Still "⏳ Pending" = Not reviewed yet
  - Changed to "✅ Approved" or "↩️ Returned" = Reviewed

---

### **Q: Can I edit a pending proposal?**
**A:** 
Currently no direct edit feature. Options:
1. Wait for chairperson to return it
2. Contact chairperson to request return
3. Submit a new proposal if urgent

---

### **Q: My proposal was returned. Do I edit the same one?**
**A:**
No. Create a NEW proposal:
1. Click "Create Revised Proposal" button
2. Opens new proposal form
3. Fill in with corrected information
4. Reference chairperson's comments
5. Submit as new proposal

## ✅ **Quick Status Check Checklist**

**☑️ Is my proposal approved?**

Check these indicators:
- [ ] Green ✅ notification in bell
- [ ] Status badge shows "✅ Approved" in My Proposals
- [ ] Event appears in Events page
- [ ] Activity log shows "[Chairperson] approved..."
- [ ] Green comment box with chairperson's approval message

**If ALL are present → APPROVED! ✅**

---

**☑️ Is my proposal returned?**

Check these indicators:
- [ ] Orange ↩️ notification in bell
- [ ] Status badge shows "↩️ Returned" in My Proposals
- [ ] Orange comment box with chairperson's feedback
- [ ] Event does NOT appear in Events page
- [ ] "Create Revised Proposal" button visible

**If present → RETURNED - needs revision ↩️**

---

**☑️ Is my proposal still pending?**

Check these indicators:
- [ ] No new notification
- [ ] Status badge shows "⏳ Pending" in My Proposals
- [ ] No chairperson comments yet
- [ ] Event does NOT appear in Events page
- [ ] Submitted date shown, no reviewed date

**If present → Still PENDING ⏳**
